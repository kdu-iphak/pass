Apps Script 배포 방법
1. Google Sheets 열기 → 확장 프로그램 → Apps Script
2. Code.gs 내용 붙여넣기 후 저장
3. 배포 → 새 배포 → 웹 앱 선택
4. 실행: 나, 액세스: 모든 사용자 → 배포
5. 생성된 URL 아래에 붙여넣기
